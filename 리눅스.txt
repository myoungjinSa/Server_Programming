-리눅스는 Unix의 소스코드를 복사한게 아니라 처음부터 unix와 같이 동작하게 새로 만든것이다.


*유명 배포판

-우분투 : desk top 용에서 많이 사용 -> UI가 잘 되어 있음
	윈도우용 게임이 실행 안됨

-Red Hat : GUI가 없음, 서버 전용 배포판, 유료, 버그 수정 해줌, 웹 서버에 특화 되어있다.

-Cent OS : Red Hat의 공짜, 아무런 지원이 없음 , 최신 기술과 최신 하드웨어에서는 호환이 안됨
	 -> 게임회사에서 많이 사용ㅇ


*VS_CODE -> Visual Studio 에서 리눅스 컴파일러로 컴파일 할 수 있음
-> 개발컴퓨터와 서버 컴퓨터가 분리 되어있음 
-> 개발 컴퓨터는 윈도우로 되어있고 vs_code를 이용해 서버 컴퓨터에 소스와 실행파일을 만듬


*리눅스 
-장점 :	1. 가격이 싸다
         	2. 운영체제 크기가 가볍다.
	3. Open Source이다. -> 커널까지 디버깅 가능
	4. 리눅스보다 윈도우가 성능이 좋지만 리눅스를 튜닝하면 게임서버를 사용하는데서 더 좋을 수 있다.
-단점 :    1. 익숙해지기 어렵다.
